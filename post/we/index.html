wallpaper_engine_动态壁纸

20190419


目前软件版本 1.1.165，steam发布于 2019-04-17

软件介绍

wallpaper engine 是最早的动态壁纸软件，Steam上卖十八块钱，也不贵，能入正就入正，毕竟steam上生态好，创意工坊有很多好看的壁纸，一直都会更新。
这里放的是破解版，不能连接Steam的，你只能本地使用。

更新说明

  • Fixed image layers that were set to be sprite sheets before support for sprite sheets was added.
  • Fixed removal button of camera paths.
  • Fixed particle instance UI options de-syncing with wallpaper and having no effect anymore.
  • Forced shader rebuilt after new effect import.
  • Added option to skip effect upgrade.
  • Added image sequence duration in filenames.
  • Added more definitive clone wallpaper management when monitor aspect ratios are incompatible.
    (看不懂英文的话，就自行用翻译工具翻译一下)

下载说明

链接: https://pan.baidu.com/s/1cVCxWnTTL59c4XejG9IXMw 提取码: 8023

文件结构说明

  • “软件本体”文件夹：里头是wallpaper_engine的这个软件；
  • “[松鼠党]动态壁纸收藏”收藏了一些动态壁纸，供你下载使用，把壁纸放到".\projects\defaultprojects"目录，关闭软件再次打开就有壁纸了；
    -“所需运行库-如果你有就不用下载”是运行wallpaper_engine必备的运行库，分别是Microsoft Visual C++ Pack 和DirectX End-User Runtimes

安装说明

如果你缺少运行库，请先安装Microsoft Visual C++ Pack 和DirectX End-User Runtimes

  1. 下载Wallpaper Engine v1.1.165.7z后，完全解压到你安装软件的目录；
  2. 运行“点我安装.exe”；

  1. 跳出首次运行窗口，把语言设置为中文；


  2. 其余设置随意，根据自己喜好来。本版本是破解版，无法使用创意工坊，所以不用管那个提示。安装主题请把下载的动态壁纸放到".\projects\defaultprojects"目录。

5. 安装过程后选择： - 运行Wallpaper32.exe（禁用UI硬件加速） - 运行Wallpaper64.exe（可能使用更多内存）

二选一，一般64位就运行Wallpaper64.exe吧
